The following people and organizations were pivotal to the development of Godhood - The Eye of Ascension.

Sun Systems Java programming language and MIDP API obviously plays no small part in the development in Godhood.

The perennial Nethack is a game of the same genre and a favorite of mine.

The Gimp was used to draw all the images in Godhood.

The IDE IntelliJ is just awesome for java development.

Room files are generated from XML definitions using the excellent JDom libraries.

My build scripts all use Ant. The j2me specific extensions provided by Antenna have also proven extremely useful.

The obfuscator Proguard made it possible to make the game of a reasonable size without having to comprimise on functionality.

Finally I'd like to thank my friends, whose support and feedback has made this game possible.

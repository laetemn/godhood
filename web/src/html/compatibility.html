Godhood - The Eye of Ascension is designed to work with most j2me 1 enabled devices. Currently the completed game requires the device to have the following specifications
  • Complete implementation of MIDP 1.0
  • Support for MIDlets of up to 64 KB
  • 50 KB, or more, of available memory2
  • Storage space for up to 30 KB of temporary game data
The game supports screens of any size, however it works best on large, color screens.

At this stage, Godhood - The Eye of Ascension does not require any network connections in order to play. In the future, remote level and score servers are expected to be added.

1. Java 2 Micro Edition
2. Any less than this and the game may run slowly or crash
